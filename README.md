# 🏢 Employee Leave Management System

A modern and responsive **Employee Leave Management System** built using **HTML, CSS, and JavaScript**. This project provides an intuitive platform for employees to submit leave requests and track their status, while administrators can efficiently manage employee records, approve or reject leave applications, and monitor leave analytics through a professional dashboard.
